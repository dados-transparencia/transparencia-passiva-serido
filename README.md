# Transparência passiva no Poder Legislativo do Seridó Potiguar

Este repositório reúne os dados, evidências documentais e instrumentos metodológicos utilizados no artigo:

**"Entre o pedido e a resposta: transparência passiva e accountability no Poder Legislativo municipal do Seridó Potiguar".**

## Conteúdo
- Planilha geral com controle das solicitações e pontuação por critério
- Prints das solicitações e respostas por Câmara Municipal
- Escala de avaliação da transparência passiva

## Observações éticas
- Os dados são de natureza pública, obtidos por meio de solicitações com base na Lei nº 12.527/2011.
- Informações pessoais foram anonimizadas.
- O repositório tem finalidade exclusivamente acadêmica.
